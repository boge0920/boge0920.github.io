Welcome to RSHadoop

Introduction

RSHadoop is a Rough Set toolkit for massive data analysis on Hadoop, developed by Junbo Zhang (SWJTU).

Papers

[1] Junbo Zhang, Tianrui Li, Da Ruan, Zizhe Gao, Chengbing Zhao, A Parallel Method for Computing Rough Set Approximations, Information Sciences, vol. 194, pp. 209-223, 2012. [Paper]

[2] Junbo Zhang, Tianrui Li, Yi Pan, Parallel Rough Set based Knowledge Acquisition using MapReduce from Big Data. ACM SIGKDD12 Big Data Mining (BigMine'12) Workshop, Beijing, China, pp. 20-27,2012. [Paper]
