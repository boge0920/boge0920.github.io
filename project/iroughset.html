Welcome to iRoughSet

Introduction

iRoughSet is an incremental learning tookit based on rough set theory, developed by Junbo Zhang (SWJTU).

Functionality

Some features currently offered include:

  • Incremental learning for complete information system [Categorical data]
  • Incremental learning for neighborhood information system [Categorical data & Numeric data]
  • Incremental learning for set-valued information system [Categorical data & Set-valued data]
  • Incremental learning for composite information system [Multiple different kinds of data, e.g., Categorical data, Numeric data, Set-valued data & Missing data]

Input Datasets

  • The Categorical & Numerical data can be downloaded from UCI (the UC Irvine Machine Learning Repository).
  • The set-valued data and the composite data can be generated randomly by the data generator, which is also developed by us and has included in the iRoughSet.
