Welcome to RSHadoop

Introduction

RSHadoop is a Rough Set toolkit for massive data analysis on Hadoop, developed by Junbo Zhang (SWJTU).

Functionality

Some features currently offered include:

  • Calculation of rough set approximations based on MapReduce
  • Rule acquisition based on MapReduce

On-going:

  • Feature selection (attribute reduction) based on MapReduce

 

Papers

  • Junbo Zhang, Tianrui Li, Da Ruan, Zizhe Gao, Chengbing Zhao, A Parallel Method for Computing Rough Set Approximations, Information Sciences, vol. 194, pp. 209-223, 2012. [Paper]
